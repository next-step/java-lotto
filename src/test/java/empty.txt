# 절차

돈 -> 로또발급 -> 당첨번호, 보너스번호 입력 -> 당첨개수 확인 -> 당첨금확인 -> 수익률 계산

# 돈 -> 로또발급
1. 구매 가능한 장수 판단
- 1000원 미만 : error
- 1000원 : 1장
- 1500원 : 1장

2. 로또 한장 발행
- 전달받은 랜덤번호를 객체에 담아 반환

- 전달받은 번호를 티켓에 담아서 반환
 - (6개 - 전달받은개수) 만큼 랜덤번호 생성
    *얘는 지금 요구사항 아님

2-1. 랜덤번호 생성
- 6개의 랜덤번호를 생성
- 전달받은 티켓에 담아서 반환
* 생성기준
 - 1-45사이
 - 랜덤번호
 - 정렬해서 반환


3. 당첨번호 + 보너스번호 발행
- 입력받은 6개의 당첨번호 > 담아서 반환
- 보너스 번호도 담아서 반환
* 생성기준
 - 1-45사이
 - 중복X
 - 정렬반환

4. 당첨개수 확인 + 보너스 당첨여부 확인
- 순위별 카운트를 저장
- 카운트가 5이면 보너스 당첨 여부를 확인

    4-1. 당첨개수별 순위 확인
    - 당첨개수별로 정해진 순위를 반환한다

*** 순위별 당첨개수를 저장해야한다.
>> 객체가 필요하겠군 ( Prize, 당첨개수 )


5. 총 당첨금 계산
- 순위별 당첨개수 * 순위별 상금 => 총 당첨금 반환

6. 수익률 계산
- 총 당첨금 / 구입비용
