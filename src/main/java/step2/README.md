# 2단계 - 로또(자동)

## 기능 요구사항
* 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
* 로또 1장의 가격은 1000원이다.

## 기능 구현 
입력
- "구입금액을 입력해 주세요."
- "지난 주 당첨 번호를 입력해 주세요."

비지니스 로직
- 로또 구매
    - 금액에 따른 구액 로또 수
- 로또 발급
    - 로또 수 에 따른 자동 번호 생성
        - 번호 6개 생성
        - 생성 길이 확인
        - 범위 확인
- 당첨 확인
    - 당첨 번호와 구매한 로또와 비교
    - 일치하는 당첨 번호 수에 따라 당첨 로또 수 생성
- 수익률 계산

출력
- 구매 수 출력 "14개를 구매했습니다"
- 구매한 로또 번호 출력
- 당첨 통계
    - 일치 수
    - 수익률
