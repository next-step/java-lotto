### 2단계 - 로또(자동)
#### + 3단계 - 로또 (2등)

#### 기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.
- 2등을 위해 추가 번호를 하나 더 추첨한다. (3단계)
- 당첨 통계에 2등도 추가해야 한다. (3단계)

#### 기능 목록
- 로또 범위 리스트 생성 기능
- 로또 생성 기능 (무작위, 오름차순, 6개 숫자)
- 사용자 입력 기능 (로또 구입금액, 지난주 당첨번호)
- 당첨 통계 생성 기능 (통계, 수익률)
- 당첨 통계 출력 기능
- 2등 추가번호 입력 기능 - 보너스번호 (3단계)
- 2등 추가번호 추첨 기능 (3단계)
- 2등 당첨 통계 출력 기능 (3단계)
