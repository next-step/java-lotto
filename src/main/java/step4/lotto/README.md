# [로또] 🚀 4단계 - 로또(수동)


## # Step4 요구사항

- [x] \1. 수동으로 구매할 로또 수를 입력받는다.
- [x] \2. 수동으로 구매할 로또 번호를 입력받는다.
- [x] \3. 수동으로 구매할 로또 번호를 바탕으로 로또를 발급한다.
- [x] \4. 남은 금액을 바탕으로 자동으로 로또를 발급한다.


## # Step3 피드백

- [ ] \1. 예외 클래스가 많아지면 관리가 조금 복잡해질 수 있다.
- [ ] \2. 입력한 뒤에 원하는 로또번호가 그대로 발급됐는지 확인하는 테스트를 추가한다.
- [ ] \3. "view" 를 분리한다.
