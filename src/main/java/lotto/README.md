# 로또

## 기능 요구사항

* 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
* 로또 1장의 가격은 1000원이다.

## 기능 구현 상세

* 로또 도메인 로직 구현
    * 뽑기 횟수 결정 기능
    * 자동 번호 뽑기 기능
    * 당첨 통계 기능
        * 등수별 당첨 수 집계 기능
        * 수익률 계산 기능
* 도메인-UI 인터페이스 DTO 구현
* UI 구현
    * inputView 구현
        * 구입 금액 입력
        * 당첨 번호 입력
    * outputView 구현
        * 당청 통계 출력
