# 🚀 2단계 - 로또(자동)
## 기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.

## TodoList

  -[x] 중복 없는 숫자 7개를 생성한다.
-[x] 당첨 번호를 입력한다.

  - [x] 6개인지 검증한다.
