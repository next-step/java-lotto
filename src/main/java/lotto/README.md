## 로또기능 요구사항

### Lotto

+ [x] 발급된 로또번호 6개가 아닌경우 에러
+ [x] 로또 번호 정렬
+ [x] 보너스 번호 일치 여부 확인
+ [x] 중복 번호가 있는지

### LottoGame

+ [x] 자동 로또 발급
+ [ ] 수동 로또 발급
+ [x] 총 수익률 확인

### LottoPrice

+ [x] 로또 구입금액 0이하일 경우 에러
+ [x] 로또 구입금액 게임횟수로 안나눠질 경우 에러

### WinningLotto

+ [x] 당첨 로또 생성
+ [x] 몇 등에 당첨되었는지 확인
+ [x] 2등 로또 당첨자

### LottoCount

+ [x] 전체 로또 갯수
+ [x] 수동 로또 갯수
+ [x] 자동 로또 갯수
+ [x] 입력한 로또 금액 보다 수동 로또 구매 횟수가 많을 시 에러
