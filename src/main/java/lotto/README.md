# 🎲 로또
## 기능 요구사항 분리
```text
✔️ 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
  - 로또 1장의 가격은 1,000원이다.

✔️ 한 장의 로또는 6개의 숫자가 있다.

✔️ 지난 주 당첨 번호를 입력 받는다.
  - 6개의 번호를 입력 받는다.

✔️ 당첨 통계를 구한다.
  - 지난 주 당첨 번호와 3개 이상 ~ 6개 이하 만큼 일치하는 로또 개수를 구한다.
    일치하는 개수에 따른 당첨 금액과 당첨된 로또의 개수를 구한다.
    예시) 3개 일치 (5,000원) - 1개
         4개 일치 (50,000원) - 0개
         5개 일치 (1,500,000원) - 0개
         6개 일치 (2,000,000,000원) - 0개

✔️ 로또 구매 금액 대비 당첨 금액을 비교해서 수익률을 구한다.
  예시) 총 수익률은 0.35입니다,
       14,000원의 로또를 구매하고 5,000원의 금액이 당첨되었을 때의 수익률
       
       수익률 = (당첨 금액) / (로또 구입 금액)
       수익률의 기준은 1이며, 1보다 낮으면 손해이고 1보다 높으면 이익이다.
```

## 협력 객체 책임 분리 (세부 역할)
```text
✔️ 입력 (InputView)
  - 로또 구입 금액을 입력 받는다.
  - 지난 주 당첨 번호를 입력한다.

✔️ 출력 (ResultView)
  - 발권된 모든 로또 정보를 출력한다.
    - 로또 한 장에 대한 번호 리스트를 출력한다.
  - 당첨 통계를 출력한다.
    - 당첨된 번호 개수에 따른 금액과 당첨된 로또 개수를 출력한다.

✔️ 로또 발권기 (LottoMachine)
  - 구입 금액 만큼의 로또를 발권한다.

✔️ 로또 번호 (LottoNumber)
  - 1부터 45까지의 번호를 관리한다.
  
✔️ 모든 로또 번호 (AllLottoNumber)
  - 로또 번호를 임의의 순서로 섞는다.
  
✔️ 로또 (LottoTicket)
  - 로또 한 장당 번호 개수인 6개의 번호를 관리한다.
  - 당첨 여부를 체크한다.

✔️ 구매한 로또 리스트 (LottoTickets)
   - 자동으로 로또 리스트를 생성한다.
   
✔️ 로또 통계 정보 (LottoStatistics)
   - 구매한 로또 리스트들의 당첨 정보를 관리한다.
```
