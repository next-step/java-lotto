## 1단계 - 문자열 계산기

### 기능 요구사항

* [x] 사용자가 문자열을 콘솔에 입력한다.
* [x] 문자열은 빈 공백일 경우 예외가 발생한다.
* [x] 콘솔에는 숫자와 사칙연산자만 입력할 수 있다.
* [x] 입력된 문자열을 바탕으로 사칙연산을 진행한다. 사칙연산은 우선순위가 없다.
* [x] (연산자의 수 - 피연산자의 수)가 절대값이 1이 아닐 경우 예외가 발생한다.
* [x] 연산 결과를 콘솔로 출력한다.
* [x] 피연산자는 숫자만 입력할 수 있다.
* [x] 연산자는 사칙연산자(+ - / *)만 입력할 수 있다.
