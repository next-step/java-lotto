### 요구사항

- [x] 1000*N원을 넣으면, N장의 로또를 구매할 수 있다.
- [x] 로또번호(1번 ~ 45번)는 랜덤으로 생성된다.
- [x] 로또하나는 6개의 로또번호로 구성된다.
- [x] 당첨번호와 하나의 로또티켓을 비교해서, 몇등인지 알려준다.
- [x] 등수에 맞는 상금이 있다.
- [x] 결과로 상금의 합과 구매 금액을 비교해서 수익률을 알려준다.
- [x] 사용자는 로또 번호를 입력해서 수동으로 로또를 구매할 수 있다.

## 도메인

### LottoNumber

- [x] 로또번호는 1~45다.

### LottoTicket

- [x] 1~45사이의 중복되지않는 번호들을 생성한다
- [x] 당첨티켓과 로또티켓을 비교한다.
- [x] LottoNumber가 포함되는지 확인할 수 있다.
- [x] LottoNumbers 입력과 일치되는 개수 반환
- [x] 6개의 로또 번호를 가진다.

### LottoPrize

- [x] 등수에 맞는 상금이 있다.
- [x] 맞춘개수와 보너스 맞췄는지를 받아, 등수를 반환한다.

### LottoCheckManager

- [x] 보너스번호가 선택된 우승 번호들에 포함되는지 확인한다
- [x] 사용자가 구매한 로또티켓과, 회차에서 선택된 우승 번호들을 비교해서, 등수를 알려준다.

### LottoResult

- [x] 구매한 로또 티켓을 받아서, 수익률을 계산한다.
- [x] 구매한 로또 티켓이 몇등인지 알려준다.

### LottoPurchaseManager

- [x] 돈을 받아, 랜덤으로 로또 티켓을 구매한다.
- [x] 돈을 받아, 수동으로 로또 티켓을 구매한다.
