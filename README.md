# 로또
##기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.
- 2등을 위해 추가 번호를 하나 더 추첨한다.
- 당첨 통계에 2등도 추가해야 한다.

## 구현 요구사항 정리
- [x] 로또 구입 금액 입력 구현
- [x] 구입 금액을 계산하는 기능 구현
- [x] 로또 번호 생성 구현
- [x] 로또 생성 및 일급 컬렉션 구현
- [x] 테스트 추가
- [x] 로또 번호 출력부 구현
- [x] 당첨 통계 입력부 구현
- [x] 당첨 통계 구현
- [x] 테스트 추가
- [x] 당첨 및 수익 출력
- [x] 리팩토링
- [x] 보너스 볼 입력 추가
- [x] 보너스 볼 유효성 검사 구현
- [x] 2등 당첨 판단 로직 구현
- [x] 테스트 추가
