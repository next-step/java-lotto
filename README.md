# 🚀 2단계 - 로또(자동)
## 기능 목록
- [ ] 입력한 금액으로 부터 로또가 몇장 구매 가능한지 반환
- [ ] 로또는 6자리 숫자 리스트를 반환
- [ ] N번 만큼 로또가 발급하는 기능
- [ ] 당첨 번호와 N 일치 하는지 반환
- [ ] 당첨 금액과 구입 금액으로 부터 수익률 반환

## 프로그래밍 요구사항
- 핵심 로직과 UI 로직을 분리한다.
- indent(들여쓰기) depth를 2단계에서 1단계로 줄여라.
- 메소드의 크기가 최대 15라인을 넘지 않도록 구현한다.
- method가 한 가지 일만 하도록 최대한 작게 만들어라.
- else를 사용하지 마라.
- 모든 로직에 단위 테스트를 구현한다.
- 자바 코드 컨벤션을 지키면서 프로그래밍 하기.
