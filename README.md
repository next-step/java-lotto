# 문자열 사칙 연산 계산기
## 입력
### 공백으로 문자열 나누기
### 빈 문자열일 경우 예외
### 숫자, 사칙 연산 기호(+, -, *, /)가 아닐 경우 예외
## 연산
### 기호별 연산 진행

---

# 로또(step2)
## 로직
### 로또 개수 확인(6개)
### 로또 구입 장수 계산
### 로또 번호 발급
### 일치 개수 세기
### 수익률 계산

## 화면 구성
### 문장 출력
#### - 구입금액
#### - 몇장 구매
#### - 구매한 로또 번호
#### - 당첨 번호
#### - 당첨 통계, 수익률
### 입력
#### - 구입금액 입력
#### - 당첨번호 입력
