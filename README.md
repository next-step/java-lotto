## 문자열 계산기

### 기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.

### 기능 목록 정리
- [ ] 로또
  - [ ] 로또 자동 생성 및 정렬
- [ ] 로또 당첨번호
  - [ ] 로또 당첨 여부 비교
- [ ] 로또 결과
  - [ ] 로또 갯수별 일치갯수 구하기
  - [ ] 총 수익률 계산
- [ ] 화면
  - [ ] input
    - 구입금액 입력
    - 당첨번호 입력
  - [ ] output
    - 구입 갯수와 로또 번호 출력
    - 통계정보 출력
