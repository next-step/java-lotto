# 로또 (2등)

## 요구사항

### 보너스 볼

- 출력: 보너스 볼을 입력해주세요
- 입력: 숫자 1개 입력
- 에러 종류
  - 숫자 범위 벗어남
    - 출력: 1 ~ 45 사이의 숫자를 입력해주세요.
  - 중복된 숫자
    - 출력: 중복된 숫자가 있습니다.
  - 숫자 아님
    - 출력: 숫자를 입력해주세요.

### 당첨 결과

- 2등 추가: 5개 일치 및 보너스 볼 일치

---

## 도메인

- BonusNumber: 보너스 번호
- EarningRate: 수익률
- Purchase: 구매 금액
- Scores: 당첨 분류 결과 리스트
  - 총 당첨 금액
- Ticket: 로또 번호 6자리
- Tickets: 구매한 티켓 리스트
- WinningNumbers: 당첨 번호 + 보너스 번호
  - 티켓 당첨 결과 분류
