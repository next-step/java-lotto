
## 요구사항
* 2등을 위해 추가번호 하나더 추첨하기
  * 당첨번호 입력 다음에 보너스볼 추가 입력
* 당첨 통계에 2등도 추가하기( 5개일치 + 보너스볼 일치)
  * 5개가 일치한 경우 보너스볼도 로또 번호 리스트에 있는지 체크
  * 

##프로그래밍 요구사항
* 모든기능 TDD
* enum 적용
* 일급 컬렉션 사용
* indent 1이하
* 15라인 이내 메서드
* else 금지 및 자바 컨벤션

## 기능 명세
* [ ] 보너스볼 입력받기
* [ ] 일치개수가 5개인경우 보너스볼과 일치하는지 검증후 일치하면 랭크 2로 반환하기
* [ ] 랭크 2등 리스트에 추가
