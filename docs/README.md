# ⚠️ 예외 처리
- [ ] 계산식 입력 형식의 유효성 검증
    - [x] 공백, null이 아닌지
    - [x] ~~모든 숫자와 사칙연산 사이에 공백 문자열이 있는지~~
    - [x] 숫자와 사칙연산 기호로만 이루어져있는지
      - 1 + 2 / 3 * 4
      - 공백으로 split했을 때 % 2 = 0 인덱스에는 숫자인지
      - 공백으로 split했을 때 % 2 = 1 인덱스에는 + / * - 중에 하나인지
