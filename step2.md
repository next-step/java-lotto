## 기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.

## 비즈니스 요구사항
- [X] 금액에 따라 로또를 발급할 수 있다.
- [X] 게임당 중복없이 6개의 번호를 가질 수 있다.
- [X] 몇개의 게임 번호가 당첨 번호와 매치되었는지 알 수 있다.
- [X] 로또 번호는 1 ~ 45 범위에서 생성 가능하다.
- [X] 로또의 매치 결과를 얻을 수 있다.
- [X] 로또의 수익률을 확인할 수 있다.
